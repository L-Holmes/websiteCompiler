**RESEARCH CONCLUSIONS ON ANIMATED FACE DESIGN**

- **Baby schema (Kindchenschema)** — Konrad Lorenz 1946: large head, round face, large low-set eyes, small nose/mouth, chubby cheeks triggers involuntary nurturing response via amygdala; applies cross-species and to objects
- **Neoteny = trust** — Mickey Mouse was deliberately made progressively more baby-like across films; bigger eyes, larger cranium, shorter snout; this pattern is documented and deliberate, not accidental
- **Eye size sweet spot** — ACM research: medium eyes = most trustworthy + appealing; *large* eyes increase eeriness; *small* eyes read as aggressive/dominant; the common "make eyes bigger = cuter" assumption breaks down past a threshold
- **Cartoon rendering > realism for appeal** — peer-reviewed finding: Pixar-style rendering rated more appealing, happier, and more attractive than photorealistic human faces; cartoon reduces eeriness and aggression perception
- **Wide face = dominant/aggressive** — facial width-to-height ratio (FWHR) consistently predicts perceived dominance; heroes in Disney films have narrower faces; villains (Farquaad, Yzma) skew Class III jaw / wider face ratios
- **Hero vs villain face formula (Disney research study)** — heroes: larger eyes, symmetrical features, Class I or II jaw profile; villains: narrower/asymmetric eyes, prominent jaw, angular features; children as young as 5 make these moral judgments from face alone
- **No mouth = projectable emotion** — Hello Kitty, Miffy; viewer assigns emotional state; broader appeal because it doesn't lock character into a single feeling
- **Dot eyes = innocence/helplessness** — black dot eyes signal vulnerability, elicit protective response; Hello Kitty exploits this maximally
- **Symmetry + warm color = trustworthy** — higher image brightness linked to higher perceived appeal; warm colors (yellow Pikachu, orange Nemo) outperform cool/neutral in likability tests
- **Head-to-body ratio 1:1** — classic kawaii/chibi proportion; maximizes perceived vulnerability and cuteness; Kirby, Pikachu, Charmander all approximate this

research (quantified patterns)

eye size:

western animation ≈ 1.8–2.2× real human ratio

japanese animation ≈ 2.5–3.4×


eye-to-face vertical placement:

realistic ≈ midline

animation ≈ slightly lower → increases perceived cuteness


nose size:

reduced by ~30–70% vs real proportions (anime extreme end)


mouth width:

widened ~10–25% → improves expression readability


symmetry:

near-perfect bilateral symmetry retained even under exaggeration


head-to-body ratio:

~1:3 → “cute / safe”

~1:5–1:7 → “competent / heroic”


feature prioritization (impact ranking from studies):

1. color contrast / palette



2. facial proportions



3. silhouette



4. texture / detail




recognition speed:

simplified faces processed faster than realistic faces (measurable ms differences in perception studies)


uncanny valley threshold:

realism increases appeal until ~70–85% realism → sharp drop if imperfect


exaggeration rule:

1–2 features exaggerated strongly, others simplified (prevents noise overload)


exposure effect:

repeated exposure to large-eye characters increases preference for large eyes in real faces

core “logical” rules used across industry

maximize eye area relative to face

minimize mid-face detail (nose/philtrum)

maintain symmetry → break only for expression

exaggerate limited features (avoid multi-axis distortion)

optimize for:

fast recognition

emotional clarity

silhouette readability


constrain realism below uncanny threshold (~70–85%)

reuse stable proportion templates (low variance systems)




---

practical takeaway (design constraint model)

face = parameter system:

eye size ↑ → empathy ↑

symmetry ↑ → attractiveness ↑

detail ↓ → clarity ↑

exaggeration ↑ → memorability ↑ (until noise threshold)


no single optimum → bounded ranges with consistent convergence patterns
